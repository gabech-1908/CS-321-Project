# FinTrack — Personal Finance Tracker

A React-based personal finance tracker UI built for CS321.

## Features

- **Dashboard** — Monthly balance overview with income, spending, and savings
- **Spending Tracker** — Manual entry, receipt upload, and bank linking (Plaid placeholder)
- **Subscriptions** — View and manage recurring payments
- **Budget** — Donut chart, category limits, and savings goal progress
- **AI Insights** — Smart tips based on spending patterns

## Running Locally

```bash
npm install
npm start
```

Opens at [http://localhost:3000](http://localhost:3000).

## Tech

- React 19 (Create React App)
- All styling is inline (no CSS framework)
- Mock data — no backend or database required
